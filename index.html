Resume

Projects

  • Weave

    Weave is an iOS app that transforms screenshots into a searchable, shareable knowledge base. It uses Cohere-powered semantic search to quickly surface relevant links and context, making static images actionable. With sub-50 ms latency and seamless mood-board sharing, Weave solves the problem of buried, hard-to-find screenshot information for 1,000+ users.

  • Menu Minded

    Menu Minded is a mobile app that analyzes restaurant menus to give personalized dish recommendations based on a user’s health goals or dietary restrictions. Built with React Native, Google Cloud Vertex AI, and MongoDB, it uses a weighted-sort algorithm to tailor suggestions and provide an easy, health-focused dining experience for over 100 users.

  • SwipeWise

    SwipeWise is a Chrome extension that recommends the best credit card to use while shopping online, maximizing cashback and rewards based on the current store. It analyzes each retailer’s offers in real time, helping users effortlessly earn the highest possible benefits.

  • Checkout Free

    Checkout Free is a mobile app that lets shoppers scan and purchase grocery items directly from their phones, eliminating checkout lines. Using computer vision for accurate item detection and seamless in-app payments, it provides a fast, frictionless grocery shopping experience.

  • GT Marketplace

    GT Marketplace is a web marketplace where Georgia Tech students can buy and sell items directly to each other. It streamlines peer-to-peer transactions with secure payments, built-in messaging, and smart search, making it easy to find interested buyers or sellers and complete deals quickly.
